WinMerge Documentation

This folder contains WinMerge's user- and developer-documentation.

User Documentation

User documentation is in Users-subfolder.

Important files in user documentation:

  • ReadMe
  • ChangeLog
  • ReleaseNotes
  • User manual in own folder note that it is XML files and needs to be processed into html files.

Developer Documentation

Developer documentation is in Developers-subfolder.

Important files in user documentation:

  • ReadMe for developers contains lots of development-related information.
  • Subversion info
  • Translation instructions
  • User manual information

Lots of developer information is in our development wiki. Information in the wiki is usually more up-to-date and more frequently updated than documentation in source tree.
