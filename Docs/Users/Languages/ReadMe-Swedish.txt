WINMERGE

WinMerge är ett Open Source filjämförelse och mergeprogram som fungerar på
alla moderna Windowsversioner. Några funktioner kräver ytterligare 
installationer eller komponenter.

Senaste versionen av WinMerge och annan WinMerge information finns på
http://winmerge.org

Quick start to WinMerge (Snabbstart för WinMerge):
Läs online-manualens Quick start-kapitel för att starta med WinMerge:
http://www.winmerge.org/2.2/manual/quickstart.html

HTML Manual:
Manual finns tillgångling online på
http://www.winmerge.org/2.2/manual/
det är också möjligtvis installerat (om detta är valt) lokalt och 
nerladdningsbart separat från http://www.winmerge.org/ (se dokumentation)

Archive Support (Arkivstöd):
WinMerge använder 7-Zip för arkivfilstöd. 7-Zip (http://www.7-zip.org)
är ett Open Source arkivprogram. För att installera arkivstöd , ladda ner
7-Zip plugininstalleraren från http://www.winmerge.org/.

Installing archive support (Installera arkivstöd):
Det är rekommenderat att installera 7-Zip programmet. Om detta inte kan göras av
någon anledning, kan 7-zip plugin installeraren installera bara krävda filer för
att stödja arkivfilstöd. Notera att det tillåter INTE självständigt användande 
av 7-Zip, bara arkivstöd för WinMerge.

Scripts Support (Skript-support):
Om du vill arbeta med skript måste du ha Windows
Scripting Host installerat. Om du får fel relaterade till dina skript var god 
och besök
http://msdn.microsoft.com/library/default.asp?url=/downloads/list/webdev.asp
för att vara säker på att din Windows Scripting Host är både uppdaterad och inte
trasig.

Bugs and feature requests (Fel och funktionsförfrågningar):
Fel och förslag på nya funktioner ska skickas till Felspårare och 
Funktionsbegäransspårare på sourceforge.net.

Bug tracker (Felspårare):
http://sourceforge.net/tracker/?group_id=13216&atid=113216
När du rapporterar ett fel, var god och tala om för oss WinMerges versionsnummer!
WinMerge-versionerna 2.2.0 och senare kan producera 'Configuration Log' från
menyn Hjälp/Konfiguration. Var god och bifoga denna information 
(som en bifogad fil) till felrapporten, den har mycket användbar information för
utvecklare.

RFE (Funktionsbegäransspårare) tracker:
