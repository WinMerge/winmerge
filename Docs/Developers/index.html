WinMerge Developer Documentation

This folder contains WinMerge's developer-documentation.

Developer Documentation

Developer documentation is in meant for people who want to compile WinMerge themselves, want to know how things work in WinMerge and possibly help with WinMerge development.

Important files:

  • ReadMe for developers is old "main" document. Contains still lots and lots of useful information.
  • Subversion info tells how to use subversion for version control, most importantly how to get latest sources.
  • Translation instructions
  • User manual information tells how to convert manual from DocBook to HTML and HTML help.
  • Options explains WinMerge's options handling.
  • Plugins contains documentation about plugins- system.
  • Merge7zInstaller documents archive support installer.
  • InnoSetup contains documentation about WinMerge's installer.
  • SVN Organization explains SVN folders.

Lots of developer information is in our development wiki. Information in the wiki is usually more up-to-date and more frequently updated than documentation in source tree.
